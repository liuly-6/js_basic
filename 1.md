特性：
- set的值是唯一的，不可重复的
- set 本身是构造函数
- 可用于去重/去重复字符串：[...new Set('ababbc')].join('')
- set的对比方式相当于===，因为可重复添加NaN，由于两个空对象不想等，因此也可以重复添加{}
- Array.form 可以将set结构转为数组
- set的遍历顺序就是插入顺序，比如使用 Set 保存一个回调函数列表，调用时就能保证按照添加顺序调用。
